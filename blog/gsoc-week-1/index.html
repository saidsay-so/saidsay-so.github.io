GSoC 2022: Week 1 report

2022-06-20
gsocgsoc2022pjdfstestrust

GSoC official coding period has started the previous week.

So far, we have the building blocks for the project. The runner is in a very basic state but, it works. We also have test collection, through the pjdfs_test_case and pjdfs_group macros. The documentation, while being a big work-in-progress, is available here. And we have the first tests written, for the chmod syscall.
